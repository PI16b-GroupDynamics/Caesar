Программа Caesar предназначена для шифрования и дешифрования текстов при помощи шифра Цезаря.

Текст может вводиться пользователем вручную в соответствующий контролл на главной форме. Также
пользователя предоставляется возможность загружать текстовые файлы для шифрования методом Цезаря. 
Для этого нужно перейти к пункту меню Файл -> Открыть на шифрование или Файл -> Открыть на расшифровку.

После успешной шифровки/дешифровки пользователь имеет возможность сохранить результат работы в текстовый файл
в нужной ему директории. Для этого нужно перейти к пункту меню Файл -> Сохранить зашифрованный или Файл -> Сохранить 
расшифрованный. 

В пункте меню Настройки предоставляется возможность изменить язык интерфейса программы. Доступны два языка: английский и русский.
Также такая возможность прдоставляется и на стартовом экране программы. 

Пункт меню Справка содержит информацию О программе, Об авторах, а также руководство данное руководство пользователю.

Помимо дешифрования текстов с известным смещением, программа предоставляет возможность взламывать шифр Цезаря при неизвестном
смещении. Для этого нужно загрузить текст на расшифровку и выбрать "Взлом шифра".

При помощи частотного анализа программа определит смещение и выведет на экран предполагаемый вариант расшифрованного
текста. В случае неудачи программа предоставит возможность пользователю изучить все варианты дешифрованного текста при всех
возможных смещениях.
